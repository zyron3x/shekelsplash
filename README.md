# 🎰 Shekelsplash

**SHFL Lottery Expected Value Calculator**

A premium crypto lottery EV calculator with a dark, casino-meets-DeFi aesthetic. Calculate expected value, ROI, and staking yields for the SHFL lottery system.

## 🚀 Live Site

Visit [shekelsplash.com](https://shekelsplash.com)

## ✨ Features

- **Real-time EV Calculations** - Instant updates as you adjust inputs
- **Division Breakdown** - Detailed analysis of all 9 lottery divisions
- **Standard vs Powerplay** - Compare ticket types and their expected values
- **Staking Analysis** - Calculate annual yields on SHFL staking
- **Interactive Charts** - Visual breakdown of pool distribution and EV contribution
- **Animated Particles** - Premium lottery ball physics simulation

## 🎨 Design

- Dark purple crypto/gambling aesthetic inspired by Shuffle.com
- Animated lottery ball particle system
- Responsive design for all screen sizes
- Premium glassmorphic UI elements

## 🛠️ Tech Stack

- Pure HTML/CSS/JavaScript
- Chart.js for data visualization
- Canvas-based particle system
- No build tools required - deploy anywhere
